## Development

Requirements:

- [Node v16+](https://nodejs.org)

Steps:

1. git clone
2. npm install
3. npm start

This should open the web app at http://localhost:3000

This project was bootstrapped with [Create React App](https://create-react-app.dev/docs/getting-started/).
